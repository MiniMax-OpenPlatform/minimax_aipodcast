## 快速开始

### 1. 环境要求

- **Python**: 3.7+
- **Node.js**: 16+
- **FFmpeg**: 音频处理必需

> 具体安装依赖方式可参考[快速开始](快速开始.md)

### 2. 启动服务

```bash
# 进入项目目录
cd minimax_aipodcast

# 启动服务
./start.sh
```

### 3. 访问应用

- **前端**: http://localhost:3000
- **后端**: http://localhost:5001

## 详细功能说明

### 第一步：配置 API Key

在页面顶部输入你的 MiniMax API Key。可在 [MiniMax 官网](https://www.minimaxi.com/) 获取。

### 第二步：输入内容

有三种方式可以提供播客内容：

#### 📝 话题文本（最简单）
- **适用场景**: 自己想聊的主题
- **操作**: 直接在文本框输入话题
- **示例**: 
  - "最近大火的电影《流浪地球3》"
  - "人工智能对教育的影响"
  - "如何培养良好的阅读习惯"
- **优点**: 最灵活，AI 会基于话题自由发挥

#### 🔗 网址链接
- **适用场景**: 分享某篇文章或网页
- **操作**: 输入完整的 URL（必须以 http:// 或 https:// 开头）
- **限制**: 某些网站有反爬虫机制可能无法访问

#### 📄 PDF 文档
- **适用场景**: 从 PDF 文档生成播客
- **支持格式**: 文本型 PDF
- **限制**: 不支持扫描版 PDF

---

### 第三步：选择音色

#### 默认音色（开箱即用）
- **Mini（女声）**: 活泼亲切的主持人风格
- **Max（男声）**: 稳重专业的搭档风格

#### 自定义音色（打造个性化播客）

##### 如何使用？
1. 选择 "自定义音色"
2. 上传音频文件（建议 10-30 秒）
3. 等待系统自动克隆

##### 音频要求
- ⏱️ **时长**: 10-30 秒最佳
- 🎤 **质量**: 清晰、无噪音
- 💾 **格式**: WAV、MP3、M4A 等常见格式
- 📏 **大小**: 建议小于 5MB

---

### 第四步：生成播客

点击 "🚀 开始生成播客" 按钮后，系统会：

1. **内容解析**（5-10 秒）- 提取和处理内容
2. **生成开场音频** - BGM + 欢迎语
3. **生成播客脚本**（流式）- 实时显示对话
4. **合成语音**（流式）- 边生成边可播放
5. **生成封面**（并行）- AI 生成漫画风格封面

---

### 第五步：下载和分享

生成完成后可以：

- **📥 下载音频** - 完整的 MP3 文件（含 BGM + 对话）
- **📝 下载脚本** - 纯文本对话内容
- **🖼️ 下载封面** - PNG 格式封面图

## 常见问题

### Q1: 后端服务无法启动？
**A**: 
1. 检查端口 5001 是否被占用
   ```bash
   lsof -i :5001
   kill -9 [PID]  # 杀死占用进程
   ```
2. 确认依赖已安装
   ```bash
   pip install -r requirements.txt
   ```

### Q2: FFmpeg 未找到？
**A**: 
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
```

### Q3: 网页解析失败？
**A**: 
- 确认网址可以在浏览器中正常打开
- 某些网站有反爬虫机制
- 推荐：复制网页文本，使用话题模式

### Q4: PDF 无法解析？
**A**: 
- 确保 PDF 是文本型（可复制文本）
- 扫描版 PDF 不支持
- 可以先提取文本，再用话题模式

### Q5: 音色克隆失败？
**A**: 
1. 检查音频质量：清晰无噪音
2. 音频时长 10-30 秒
3. 使用常见格式（WAV、MP3）

### Q6: 生成的播客没有声音？
**A**: 
- 检查是否安装了 FFmpeg
- 查看后端日志了解具体错误

## 最佳实践

### 🎯 内容准备

**好的话题示例**：
```
✅ "分析一下最近热映的科幻电影《流浪地球3》的特效和剧情"
✅ "探讨远程办公对工作效率的影响"
✅ "介绍5个提升编程能力的实用技巧"
```

**不好的话题示例**：
```
❌ "电影"（太简短，信息不足）
❌ "好"（无意义）
```

### 🎤 音色录制

**好的录音内容**：
- 朗读一段自然的话（15-20 秒）
- 语速正常，吐字清晰
- 情绪自然，不要太平淡
- 环境安静，无背景噪音

### ⚡ 优化生成速度

1. **话题模式最快** - 无需解析网页或 PDF
2. **内容长度适中** - 500-2000 字效果最好
3. **首次使用默认音色** - 跳过克隆时间
