# ⚡ 快速开始 - 5分钟上手指南

## 🎯 目标

5分钟内生成你的第一个AI播客！

---

## 前置要求

在开始之前，请确保已安装以下依赖：

- **Python 3.7+**
- **Node.js 16+**
- **FFmpeg**（音频处理必需）


## 安装依赖

### 安装 FFmpeg

```bash
brew install ffmpeg
```


### 进入项目目录

```bash
cd minimax_aipodcast
```

### 安装 Python 依赖

```bash
pip install -r requirements.txt
```

### 安装 Node.js 依赖

```bash
cd frontend && npm install && cd ..
```

---

## 步骤 1️⃣：启动服务（1分钟）


### 启动服务

```bash
chmod +x start.sh
./start.sh
```

**看到这样的输出说明启动成功**：
```
✅ 服务启动成功！

📍 后端地址: http://localhost:5001
📍 前端地址: http://localhost:3000
```

---

## 步骤 2️⃣：打开网页（10秒）

在浏览器中打开：
```
http://localhost:3000
```

你会看到一个现代化的 React 前端页面 🎨

---

## 步骤 3️⃣：配置 API Key

1. 在页面顶部的 **API Key 配置** 区域
2. 输入你的 MiniMax API Key
3. 可在 [MiniMax 官网](https://www.minimaxi.com/) 获取

---

## 步骤 4️⃣：输入话题（30秒）

### "输入内容" 区域

支持三种输入方式：
- **💬 话题文本** - 直接输入想聊的话题
- **🔗 网址链接** - 输入文章 URL
- **📄 上传 PDF** - 上传 PDF 文档

**推荐的第一次尝试**：
```
最近大火的科幻电影《流浪地球3》讲述了人类面对太阳危机，
带着地球一起逃离太阳系的史诗故事。影片的特效震撼，
剧情感人，展现了人类的勇气和智慧。
```

或者简单点：
```
分享一下如何培养良好的阅读习惯
```

---

## 步骤 5️⃣：选择音色（10秒）

### "选择音色" 区域

**第一次尝试，建议使用默认音色**：
- Speaker 1: Mini（女声）
- Speaker 2: Max（男声）

> 💡 熟悉流程后，再尝试自定义音色

---

## 步骤 6️⃣：生成播客（3分钟）

### 点击大按钮

点击：
```
🚀 开始生成播客
```

### 观察生成过程

你会看到：

**1. 进度状态（实时更新）**
```
正在解析输入内容...
正在准备音色...
正在生成播客脚本和封面...
播客生成完成！
```

**2. 对话脚本实时显示**
- 每句对话会实时出现

**3. 渐进式音频播放**
- 边生成边可以播放

**4. 封面图片出现**
- 漫画风格封面

**5. 底部显示 Trace ID**
- 每个 API 调用都有记录

---

## 步骤 7️⃣：下载分享（1分钟）

### 生成完成后

看到下载按钮区域，你可以：

1. **⬇️ 下载音频** - 完整的 MP3 文件
2. **⬇️ 下载脚本** - 纯文本对话脚本
3. **⬇️ 下载封面** - 播客封面图片

## ❓ 遇到问题？

### 服务启动失败

```bash
# 检查端口是否被占用
lsof -i :5001
lsof -i :3000

# 如果有进程占用，杀死它
kill -9 [PID]

# 重新启动
./start.sh
```

### FFmpeg 未安装

```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
```

### 网页无法访问

- 确认服务正在运行
- 尝试刷新页面
- 检查浏览器控制台是否有错误
