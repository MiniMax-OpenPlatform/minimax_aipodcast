## ✨ 功能特性详解

### 1. 内容输入

#### ✅ 话题模式
- 直接输入话题文本
- 适合快速创作
- 灵活度最高

#### ✅ URL 模式
- 支持网页链接解析
- 自动提取正文内容
- 过滤脚本和样式

#### ✅ PDF 模式
- 支持文本型 PDF
- 最大 10MB 限制
- 自动提取文本

### 2. 音色定制

#### ✅ 默认音色
- **Mini**: 女声，活泼亲切
- **Max**: 男声，稳重专业

#### ✅ 自定义音色
- 支持上传音频文件克隆
- 推荐音频时长 10-30 秒
- 支持 WAV、MP3 等格式

### 3. 播客生成

#### ✅ 脚本生成
- 使用 MiniMax M2-Preview 模型
- 流式生成，实时显示
- 自然对话风格

#### ✅ 语音合成
- 使用 MiniMax TTS 2.5 HD
- 高质量语音输出
- 流式合成

#### ✅ 封面生成
- 使用 MiniMax image-01-live
- 漫画风格
- 1:1 比例

### 4. 播放与下载

#### ✅ 渐进式播放
- 双缓冲播放器
- 边生成边可播放
- 无缝切换

#### ✅ 下载功能
- 音频下载（MP3）
- 脚本下载（TXT）
- 封面下载（PNG）

## 🔍 API 列表

| 端点 | 方法 | 说明 |
|------|------|------|
| `/health` | GET | 健康检查 |
| `/api/default-voices` | GET | 获取默认音色 |
| `/api/generate_podcast` | POST | 生成播客（SSE） |
| `/api/upload_audio` | POST | 上传音频文件 |
| `/download/audio/<filename>` | GET | 下载音频 |
| `/download/script/<filename>` | GET | 下载脚本 |
| `/download/cover` | GET | 下载封面 |
| `/static/<filename>` | GET | 静态文件（BGM） |
